# WebCompApp

The app demonstrates of microfrontend implementation using Angular on the TikTok clone.

## Installling dependencies

`npm i`

## Run Shell (port 3000) and Feed (port 5000) app

`npm run start`

## Run Feed app

`npm run start:feed`

## Build Shell app

`npm run build:shell`

## Build Feed app

`npm run build:feed`

## Enjoy!
